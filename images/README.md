# Images Directory

Please add your images to the following folders:

## Directory Structure

```
images/
├── gallery/
│   ├── business-cards.jpg
│   ├── awards.jpg
│   ├── packaging.jpg
│   ├── tshirts.jpg
│   ├── signage.jpg
│   └── promotional.jpg
├── services/
│   ├── offset-printing.jpg
│   ├── digital-printing.jpg
│   ├── sublimation.jpg
│   ├── uv-printing.jpg
│   ├── dtf.jpg
│   └── crystal.jpg
└── hero-background.jpg
```

## Image Recommendations

### Gallery Images (gallery/)
- **business-cards.jpg**: High-quality business card samples
- **awards.jpg**: Crystal awards and trophies
- **packaging.jpg**: Custom packaging boxes and gift boxes
- **tshirts.jpg**: T-shirt printing samples
- **signage.jpg**: Banners, flex signage, and acrylic signs
- **promotional.jpg**: Promotional products (mugs, pens, etc.)

### Service Images (services/)
- **offset-printing.jpg**: Offset printing equipment or samples
- **digital-printing.jpg**: Digital printing samples
- **sublimation.jpg**: Sublimation printing examples
- **uv-printing.jpg**: UV printing samples
- **dtf.jpg**: DTF printing examples
- **crystal.jpg**: Crystal momento products

### Hero Background
- **hero-background.jpg**: Professional background image (1920x1080 recommended)

## Image Specifications

- **Format**: JPG or PNG
- **Gallery Images**: Minimum 800x600px, recommended 1200x900px
- **Service Images**: Minimum 400x300px, recommended 600x450px
- **Hero Background**: 1920x1080px or larger
- **Optimization**: Compress images for web to ensure fast loading

## Note

If images are not available, the website will display placeholder icons automatically.
